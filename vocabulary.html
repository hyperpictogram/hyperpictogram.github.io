Vocabulary

Vocabulary URI: http://pictogram.tokyo/vocabulary

Class: Pictogram

name
A name of the pictogram; a string literal; multilingualized with `lang` attributes.
shortDescription
A short desciption of the pictogram; a string literal; multilingualized with `lang` attributes.
image
A URI of the image of the pictogram; the image should be SVG.
longDescription
A long description of the pictogram; a string literal; multilingualized with `lang` attributes.
author
A name of the author of the pictogram; a string literal; multilingualized with `lang` attributes.
source
A source of the pictogram; a string literal; multilingualized with `lang` attributes.
sourceURI
A URI of the source of the pictogram.
permission
A URI of the permission/license; Creative Commons, for instance.
